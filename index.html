Project Overview


We are creating a calculator web app. We will be using JavaScript. The purpose of this project is to learn the basics of web development, and see if we can make a functioning app that looks nice. We are also going to try to see if we can also experiment with web advertising with Google Ads.

We are going to start with familiarizing ourselves with JavaScript, by writing a few other basic web programs. We are going to experiment with buttons and all of the web elements. Then, we are going to try to build the app, then try to expose an API endpoint that we can send requests to. Finally, we are going to experiment with web advertising, and try to see how websites can become profitable.
